## 🚀 Getting Started

### Prerequisites
- GCC compiler installed
- C99 standard support

### Compilation

Compile a single file:
```bash
gcc -o program_name program_name.c
```

Compile and run:
```bash
gcc -o program_name program_name.c && ./program_name
```

Compile all files in a directory:
```bash
gcc C/Arrays/*.c -o arrays_suite
```

### Running Programs

Execute a compiled program:
```bash
./program_name
```

Most programs are interactive and accept input via `scanf()` or `fgets()`.

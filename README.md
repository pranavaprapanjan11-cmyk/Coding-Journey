# Coding Journey

A structured repository containing solved programming problems, algorithms, and concepts in C, C++, and Java. This repository represents a personal learning journey starting from basic loops and patterns to advanced data structures and algorithm designs.

## Languages Used

* **C**: Low-level programming, bitwise operations, memory management, array manipulation, and string operations.
* **C++**: Object-oriented programming, standard template library (STL), dynamic programming, and algorithm optimization.
* **Java**: Application logic, console-based systems, and OOP-based project structures.

## How to Run Programs

### C Programs
To compile and run any C program (e.g., `Reverse_Array.c`):
```bash
gcc C/Arrays/Reverse_Array.c -o reverse_array
./reverse_array
```

### C++ Programs
To compile and run any C++ program (e.g., `hcf.cpp`):
```bash
g++ CPP/hcf.cpp -o hcf
./hcf
```

### Java Programs
To compile and run Java programs:
```bash
javac Java/StudentManagementSystem.java
java -cp Java StudentManagementSystem
```
